# UrbanFlow AI — Project Structure

## web

Frontend-приложение.

Отвечает за:

- 3D-город;
- отображение дорог, зданий, машин и пешеходов;
- редактор дорог и событий;
- панели управления;
- визуальную аналитику;
- heatmap пробок;
- compare mode.

## server

Backend-приложение.

Отвечает за:

- API;
- загрузку данных из OpenStreetMap;
- подготовку карты для симуляции;
- запуск симуляции;
- хранение состояния;
- WebSocket-стрим состояния во frontend;
- связь с AI-модулем.

## ai

AI-модуль.

Отвечает за:

- RL-среду;
- агентов перекрестков;
- graph neural network;
- reward;
- обучение модели;
- оценку модели;
- сохранение весов.

## tests

Тесты проекта.

## scripts

Локальные скрипты для запуска, проверки и подготовки данных.

## data

Локальные данные проекта.

В git не попадают реальные данные, модели и результаты запусков.
